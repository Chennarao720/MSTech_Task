# MSTech_Task
Ecommerce React Project

This is a modern *Ecommerce web app* built with *React + Vite* and styled using *Tailwind CSS*.

---

 Features:
-- Beautiful and responsive UI
-- Product listing and filtering
-- Shopping cart functionality
-- React Router-based navigation
-- JSON Server backend for mock data
-- Fake Store API

---

 Tech Stack
-- React
-- Vite
-- Tailwind CSS
-- Fetch
-- React Router DOM(Browser router way)(dynamic and nested routing)
-- Framer Motion
-- JSON Server
-- React Tostify
